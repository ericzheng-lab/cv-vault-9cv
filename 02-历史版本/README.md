# 历史版本归档

> 所有已投递岗位的材料统一归档在此

## 目录结构

每个岗位一个文件夹，命名格式：`{公司}_{岗位}-{日期}`

```
02-历史版本/
├── GOOGLE_Producer-Deepmind-2026-04-06/
│   ├── JD.md              # JD 原文
│   ├── analysis.md        # 岗位分析
│   ├── resume.html        # 定制简历
│   ├── cover-letter.html  # 定制求职信
│   └── resume.pdf         # 最终投递版
├── EMERSON_AVP-Executive-Producer/
│   └── ...
└── SEMAFOR_Executive-Producer/
    └── ...
```

## 归档规则

1. **JD 必须包含** - 从 `05_JD/` 复制
2. **生成内容必须包含** - 从 `03-生成中/` 复制
3. **最终 PDF 必须包含** - 投递后导出
4. **所有材料放在一起** - 方便面试前回顾

## 工作流程

1. 在 `03-生成中/` 完成定制
2. 投递后，复制到 `02-历史版本/{公司}_{岗位}/`
3. 保留所有版本（analysis-v1, v2, v3...）
